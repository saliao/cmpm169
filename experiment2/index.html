Experiment 2 - Vector Art, Animation & Interactivity

Imitate

I picked a manga panel from OPM. The mysterious figure to draw in p5.js

I did not find any code for imitating the figure so I used ChatGPT to get a stick figure as reference and start from there

Integrate

After learning from the existing p5.js library and reference pages, I drew the figure as best as I could and then looked open code from opening processing: https://openprocessing.org/sketch/2036451.

I Integrated this code into my project by using it as my background and also learning how to modify it and change the blurr and shadows on my figure.

Innovate

Finally, made it more interactive and animated by looking up how to redraw and also use interactive functions.

I used these sources to get it going: https://stackoverflow.com/questions/54153201/how-do-you-redraw-an-object-created-from-a-class-in-p5-js-javascript

https://p5js.org/reference/#/p5/mouseMoved

Now when we move the mouse, the figure changes transparncy and the stars change too
